# Intermediate Value Theorem 
Let $I = [a, b]$ be an interval and $f: I \rightarrow \mathbb{R}$ be continuous on $I$. If $f(a) < 0 < f(b)$ or $f(a) > 0 > f(b)$, then there exists a number $c \in (a, b)$ such that $f(c) = 0$.
